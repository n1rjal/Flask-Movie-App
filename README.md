## Installation

OS X & Linux:

```sh

sudo apt install git 
sudo apt install python3
sudo apt get python3-pip
pip3 install pipenv
git clone https://github.com/Aju100/Flask-Movie-App.git
cd Flask-Movie-App
pipenv install
python3 app.py
```

Windows:
You can download git and python3 and install pip as well as pipenv package.
```sh
git clone https://github.com/Aju100/Flask-Movie-App.git
cd Flask-Movie-App
pipenv install
python app.py
```
